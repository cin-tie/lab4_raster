# Raster Algorithms Demo

Веб-приложение для демонстрации растровых алгоритмов:
- Step (пошаговый)
- DDA
- Bresenham (линия)
- Bresenham (окружность)

Реализовано: HTML5 canvas + JS (frontend), Flask (backend — минимально, только чтобы раздать статический сайт).
